113        // One type of button is a joystick button which is any button on a joystick.
114        // You create one by telling it which joystick it's on and which button
115        // number it is.
116        // Joystick stick = new Joystick(port);
117        // Button button = new JoystickButton(stick, buttonNumber);
118        
119        // Another type of button you can create is a DigitalIOButton, which is
120        // a button or switch hooked up to the cypress module. These are useful if
121        // you want to build a customized operator interface.
122        // Button button = new DigitalIOButton(1);
123        
124        // There are a few additional built in buttons you can use. Additionally,
125        // by subclassing Button you can create custom triggers and bind those to
126        // commands the same as any other Button.
127        
128        //// TRIGGERING COMMANDS WITH BUTTONS
129        // Once you have a button, it's trivial to bind it to a button in one of
130        // three ways:
131

133        // until it is finished as determined by it's isFinished method.
134        // button.whenPressed(new ExampleCommand());
135        
136        // Run the command while the button is being held down and interrupt it once
137        // the button is released.
